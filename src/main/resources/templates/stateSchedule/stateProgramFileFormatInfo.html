Структура файла

Для удачного формрирования "Плана-графика" необходимо чтобы файл содержал следующие элементы:

  • Колонтитул с указанием года: "Казань, 2018"
  • Одна таблица содержащая инфромацию о "Планах-графиках"

Описание таблицы "План-график"

Структура файла:

  • Название колонок соответсвуют основным полям "Плана-графика"
  • "Планы-графики" разделены по месяцам. Каждый новый месяц записан в строке, ячейки которой склеены в одну. Месяцы введены кирилицей, полностью. Пример: Март, Февраль, Апрель...

Названия колонок и формат заполнения

В следующей таблице приведены названия основых полей "Плана-графика". В файле название колонки должно содержать данную последовательность символов. Возможны дополнительные символы в начале строки и в конце.

Название дополнительной профессиональной программы Название дополнительной профессиональной программы Управление развитием образовательных организаций в условиях модернизации образования
Форма обучения Форма обучения (очная, заочная, очно-заочная) Возможны только следующие значения: Очное/Заочное/Очное-заочное
Форма реализация Форма реализация (модульная, интегрированная, корпоративная, командная) Возможны только следующие значения: Модульная с использованием ДОТ/Модульная
Количество слушателей Кол-во слушателей, всего чел./групп 21/1 (Символ "/" обязателен, первое число кол-во человек, второе - кол-во групп)
